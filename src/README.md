\# Variational Autoencoder with Convolutional Dictionary Learning (VAE-CDL)



This project implements a hybrid \*\*Variational Autoencoder (VAE)\*\* combined with \*\*Convolutional Dictionary Learning (CDL)\*\* for robust representation learning, sparse feature extraction, and generative modeling.



The framework integrates probabilistic latent variable modeling with structured sparse coding to improve reconstruction quality, interpretability, and robustness.



---



\## Overview



Variational Autoencoders learn probabilistic latent representations, while Convolutional Dictionary Learning extracts structured sparse features. Combining both enables:



\- Better reconstruction quality  

\- Sparse and interpretable latent representations  

\- Improved robustness to noise  

\- Enhanced generative capability
